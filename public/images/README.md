# Image Assets

Place your images in this directory:

## Required Images

1. **profile.jpg** - Your professional photo (recommended 400x400px or larger, square)
2. **logo.png** - Your logo (recommended 40x40px to 80x80px)

## Optional Images

You can also add:
- Project screenshots
- Service icons
- Background images
- Any other custom graphics

## Image Optimization Tips

- Use WebP format for better compression
- Keep file sizes under 500KB for photos
- Use transparent PNG for logos
- Optimize images before uploading (use tools like TinyPNG or Squoosh)
